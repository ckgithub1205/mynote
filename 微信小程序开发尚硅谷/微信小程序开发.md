# 2.开发小程序项目
## 1.前言
(1)小程序没有dom对象，一切基于组件化。
（2）小程序的四个重要的文件
         a.      *.js
         b.      *.wxml     => view样式    =>   html
         c.       *.wxss     => view样式     =>   css
         d.       *.json      => view样式    =>  json文件
（3）储备知识
        a.理解事件机制
        b.理解组件化
        c.理解数据绑定
        d. Flex布局
        e.移动端适配方案
